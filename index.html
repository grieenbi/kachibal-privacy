까치발 개인정보처리방침

시행일: 2026년 6월 9일  |  최종 수정: 2026년 6월 9일

그리엔비(이하 "회사")는 매주 릴레이 에세이를 이어 쓰는 서비스 까치발을 운영하며, 이용자의 개인정보를 소중히 여깁니다. 본 방침은 앱 이용 과정에서 수집·이용·보관·파기되는 개인정보에 관해 안내합니다.

1. 수집하는 개인정보 항목

  • 회원가입·로그인: 소셜 로그인(Google, 카카오)을 통해 고유 식별자(UID)를 수집합니다. 이메일 주소 및 소셜 계정 표시 이름은 저장하지 않습니다.
  • 닉네임: 서비스 가입 시 직접 입력한 닉네임
  • 프로필 사진: 직접 업로드한 사진 (선택, Base64 인코딩 후 저장)
  • 작성 콘텐츠: 에세이(발자국), 댓글, 쪽지
  • 활동 데이터: 좋아요, 팔로우 관계
  • 기기 정보: 푸시 알림 발송을 위한 FCM 토큰

2. 개인정보 수집·이용 목적

  • 서비스 제공 및 회원 관리
  • 릴레이 에세이 작성·열람·공유 기능 제공
  • 좋아요·댓글·쪽지 등 이용자 간 상호작용 기능 제공
  • 월간 단행본 발행 및 랭킹 집계
  • 푸시 알림(좋아요, 댓글, 공지 등) 발송

3. 개인정보 보유·이용 기간

회원 탈퇴 시 수집된 모든 개인정보(에세이, 댓글, 쪽지, 팔로우, 프로필 등)를 즉시 삭제합니다. 단, 관련 법령에 따라 보존이 필요한 경우 해당 기간 동안 보관 후 삭제합니다.

4. 개인정보의 제3자 제공

회사는 이용자의 개인정보를 원칙적으로 외부에 제공하지 않습니다. 다만, 서비스 운영을 위해 아래 외부 서비스를 활용합니다.

  • Google Firebase (Firebase Auth, Cloud Firestore, Cloud Functions, FCM) — 인증·데이터베이스·알림 처리
  • 카카오 로그인 — 소셜 로그인 인증

각 서비스의 개인정보 처리방침은 해당 서비스 운영사 정책을 따릅니다.

5. 개인정보 파기 절차 및 방법

  • 앱 내 '회원 탈퇴' 기능을 통해 즉시 파기 요청 가능
  • 탈퇴 시: 에세이, 댓글, 쪽지, 팔로우, 유저 정보, Firebase 인증 계정 전체 삭제
  • 전자적 파일은 복구 불가능한 방법으로 삭제

6. 이용자의 권리

이용자는 언제든지 아래 권리를 행사할 수 있습니다.

  • 개인정보 열람 요청
  • 개인정보 수정 (앱 내 '내 정보' 탭에서 직접 변경)
  • 개인정보 삭제 및 처리 정지 (앱 내 '회원 탈퇴' 기능 이용)

7. 미성년자 보호
